ক্রিয়েটিভ ফ্রন্টএন্ড

ওয়েব ডেভলপমেন্টের জগতে HTML, CSS & Javascript সাহায্যে যারা রেসপনসিভ ওয়েব ডিজাইন ইন্টারেক্টিভ অ্যানিমেশন এবং কাস্টম ওয়েব অ্যাপ্লিকেশন দক্ষ হয়ে নিজের ক্যারিয়ার ক্যারিয়ার গড়ার পরিকল্পনা করছেন তাদের জন্য সাজানো ক্রিয়েটিভ ফ্রন্ট এন্ড নামক এই কোর্সটি।

এই কোর্সের মধ্যে যা যা শিখানো হবে-

  • কম্পিউটারের বেসিক কিছু স্কিল দেখানো হবে যেগুলো দিয়ে খুব সহজেই আমরা ডেভেলপমেন্টের জগতে পা রাখতে পারব।

  • এইচটিএমএল, সিএসএস দিয়ে একটি ল্যান্ডিং পেজ ও একটি পার্সোনাল পোর্টফোলিও তৈরি শেখানো হবে।

  • বুষ্টঅ্যাপ ফ্রেমওয়ার্ক তৈরি করে রেস্পন্সিভ গ্রিড এর মাধ্যমে একটি রেস্টুরেন্ট এবং রিটেল ওয়েবসাইট তৈরি করা শেখানো হবে।

  • জাভাস্ক্রিপ্ট এর খুটিনাটি সবকিছু শিখে জাভাস্ক্রিপ্টের বিভিন্ন প্রবলেমের সলিউশন করা দেখানো হবে।

  • API এবং JSON এর সাহায্যে লিস্টিং ওয়েবসাইট এবং ইমেজ ডাউনলোডের ওয়েবসাইট তৈরি করা শেখানো হবে।

  • এই কোর্সের মাধ্যমে HTML, CSS, SASS, BOOTSTRAP 5, JAVASCRIPT এর মাধ্যমে সর্বমোট ১২ টি প্রোজেক্ট ও ৩০ টি এসাইনমেন্ট কমপ্লিট করানো হবে।
